# 煮食插件
### 修改遊戲內煮食玩法
### 原版遊戲只需要將食物與燃料放入熔爐即可
### 此插件添加新熔爐方塊、食譜系統、新的煮食動畫，並將每次新發現的食譜記錄到資料庫
